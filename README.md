# telegram-bot
Развлекательный Telegram-бот, написанный на Python с использованием библиотеки `pyTelegramBotAPI` и СУБД `SQLite`. Бот включает в себя систему регистрации пользователей, кастомную игровую валюту (кредиты), магазин подписок и встроенные мини-игры.

## 🚀 Функционал
* **Система авторизации:** Регистрация пользователей с сохранением данных и уникальных игровых "погонял" в БД.
* **Игровая экономика:** Начисление кредитов за активность, возможность перевода валюты между пользователями.
* **Мини-игры:** * *Бак Дайс (Кости)* — кастомная игра со ставками и динамическим подсчетом очков.
  * *Угадай персонажа* — текстовая викторина на знание поп-культуры и интернет-мемов.
  * *Крокодил* — пошаговая игра для групп с назначением ведущего.
* **Магазин:** Покупка подписки на премиум-контент (мемы) за внутриигровые кредиты.
* **Админ-панель:** Возможность ручного начисления кредитов администратором.

## 🛠 Технологии
* **Язык:** Python 3.x
* **Библиотека бота:** `pyTelegramBotAPI` (telebot)
* **База данных:** SQLite3 (архитектура вынесена в отдельный класс `UserDatabase`)
* **Логирование:** Встроенный модуль `logging` для отслеживания ошибок и действий пользователей.
